Firebase verbinden

Einmalige Einrichtung (ca. 5 Min.) — danach synchronisieren sich deine Aufgaben automatisch zwischen allen Geräten.

  1. Öffne console.firebase.google.com und melde dich an.
  2. Klicke „Projekt hinzufügen" → Name vergeben → erstellen.
  3. Links: „Erstellen" → „Realtime Database" → „Datenbank erstellen" → Standort wählen → „Im Testmodus starten".
  4. Links oben: Zahnrad ⚙️ → „Projekteinstellungen".
  5. Runterscrollen zu „Ihre Apps" → Web-Symbol </> klicken → Name vergeben → „App registrieren".
  6. Den firebaseConfig = { ... } Block komplett kopieren und unten einfügen.
Ungültiges Format. Bitte den vollständigen firebaseConfig-Block einfügen (mit apiKey, databaseURL und projectId).
