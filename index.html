Firebase verbinden

Einmalige Einrichtung — danach synchronisiert alles automatisch.

  1. Öffne console.firebase.google.com
  2. „Projekt hinzufügen" → Name → erstellen
  3. „Erstellen" → „Realtime Database" → „Datenbank erstellen" → Testmodus
  4. ⚙️ → „Projekteinstellungen" → runterscrollen
  5. Web-App </> registrieren → firebaseConfig kopieren
Ungültig. Bitte den vollständigen firebaseConfig einfügen.
